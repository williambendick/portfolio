Projects

Question DB

This is the database repo for an application I am developing called Question. Question is a diagnostic tool and symptom tracker used for managing the process of problem solving.

The repo contains a schema description, ER diagrams, screenshots of the application UI, and SQL scripts to build, populate and query the database.

Audiopath

Audiopath is a tool for creating interactive visualizations from sound. It synchronizes the beats in audio tracks with motion in animations.

This version of the application is a prototype, and I am continuing to develop and add functionality to it.
